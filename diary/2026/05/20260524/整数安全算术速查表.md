Rust 整数安全算术速查表

覆盖四类常用运算变体，附行为、示例、适用场景，日常编码直接对照选用

基础区分总览

运算类型	溢出表现	返回值	适用场景
原生 -/+/*	Debug 报错 panic，Release 循环环绕	原值类型	百分百确定无溢出
saturating_*	触及边界就截断卡死	原值类型	计数、长度、配额、缓冲区边界
checked_*	溢出返回 None	Option<原值>	需要手动判断溢出分支
wrapping_*	二进制循环环绕	原值类型	哈希、校验和、底层位运算

---

1. 饱和运算 `saturating`

溢出卡在类型最值，无报错

加减乘

```rust
// 无符号数，下溢卡0，上溢卡最大值
assert_eq!(5u32.saturating_sub(10), 0);
assert_eq!(u8::MAX.saturating_add(1), u8::MAX);
assert_eq!(20u8.saturating_mul(10), u8::MAX);

// 有符号数，溢出卡最值
assert_eq!(i32::MIN.saturating_sub(5), i32::MIN);
assert_eq!(i32::MAX.saturating_add(2), i32::MAX);
```

常用方法

- `saturating_add` 饱和加

- `saturating_sub` 饱和减

- `saturating_mul` 饱和乘

---

2. 校验运算 `checked`

溢出返回 `None`，正常返回 `Some(val)`，适合分支判断

```rust
// 减法溢出
assert_eq!(5u32.checked_sub(10), None);
// 正常运算
assert_eq!(10u32.checked_sub(3), Some(7));

// 加法溢出
assert_eq!(u8::MAX.checked_add(1), None);
```

业务写法示例

```rust
let a = 10usize;
let b = 20usize;
match a.checked_sub(b) {
    Some(res) => println!("差值：{}", res),
    None => println!("数值不足，溢出"),
}
```

常用方法

- `checked_add` / `checked_sub` / `checked_mul` / `checked_div`

---

3. 环绕运算 `wrapping`

按二进制位数循环截断，数学溢出但二进制合法

```rust
// u8 范围0~255，255+1 环绕为0
assert_eq!(u8::MAX.wrapping_add(1), 0);
// 0-1 环绕为最大值
assert_eq!(0u8.wrapping_sub(1), u8::MAX);
```

常用场景：简单哈希、序列号自增、协议校验计算

---

4. 配套辅助运算

abs 绝对值（有符号专属）

溢出保护：i32::MIN.abs() 单独溢出，可用saturating_abs

```rust
assert_eq!((-5i32).abs(), 5);
assert_eq!(i32::MIN.saturating_abs(), i32::MAX);
```

clamp 范围限定

把数值约束在指定区间

```rust
let n = 100u32;
// 限制 10~50 之间
assert_eq!(n.clamp(10, 50), 50);
```

---

高频场景选型建议

1. 缓冲区长度、任务计数、配额扣减 → saturating_sub/add
2. 金额、索引、关键数值校验 → checked_* 判空兜底
3. 哈希、校验码、底层二进制运算 → wrapping_*
4. 边界约束数值 → clamp
5. 逻辑确定绝不会溢出 → 直接原生运算符

---

极简记忆口诀

- 饱和卡死不越界，计数边界优先用
- 校验返回 Option，溢出分支好处理
- 环绕循环二进制，哈希位算最合适
