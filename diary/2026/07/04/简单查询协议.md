# 简单查询协议

客户端发送一条 **Query ('Q')** 消息：

```text
'Q'
Int32   长度
String  查询字符串 (null 结尾)
```

服务器可回复任意多条下列消息，最后以 `ReadyForQuery` 结束：

- **RowDescription ('T')** – 描述结果集列

- **DataRow ('D')** – 一行数据

- **CommandComplete ('C')** – 命令执行完成

- **EmptyQueryResponse ('I')** – 空查询

- **ErrorResponse ('E')** – 错误

- **NoticeResponse ('N')** – 注意/警告

- **CopyInResponse ('G')** / **CopyOutResponse ('H')** 等

## RowDescription ('T') 格式：

```text
'T'
Int32   长度
Int16   列数
对每一列：
  String  列名 (null 结尾)
  Int32   表 OID (可为 0)
  Int16   列属性号
  Int32   数据类型 OID
  Int16   类型长度 (例如 VARCHAR 的 -1)
  Int32   类型修饰符 (typmod)
  Int16   格式码 (0 文本, 1 二进制)
```

## DataRow ('D') 格式：

```text
'D'
Int32   长度
Int16   列数
对每一列：
  Int32   值长度（字节数，-1 表示 NULL）
  ByteN   值数据（文本格式无 null 结束符）
```

## CommandComplete ('C') 格式：

```text
'C'
Int32   长度
String  命令标签 (null 结尾)
```

标签示例："SELECT 3", "INSERT 0 1", "DELETE 5" 等。
