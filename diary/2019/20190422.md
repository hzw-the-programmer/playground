# redux

  listener由subscribe注册。

* redux的store源码用slice复制数组。用indexOf和splice删除元素。用push添加元素。

# js
* js的array在赋值时是引用赋值没有值拷贝（代码参见nodejs/array/assignment.js）。PHP是值拷贝。
* js的array在做相等判断时，即===时，只看引用是否相等，不比较数组内容(代码参见nodejs/array/equal.js)。
